## Course Description

This course provides a technical overview of the practical fundamentals of developing web applications. Students learn modern concepts, terminology, and popular frameworks to develop full-stack web applications.

**Technologies covered:** HTML/CSS, Node, React, TypeScript, Webpack, and deployment bundling.

**Major project:** Build your own email web application.

**Duration:** 1.5 weeks

## Prerequisites

- At least hello-world level experience with HTML/CSS/JavaScript (not strictly required, but helpful).

## Learning Objectives

1. Foundational front-end skills: HTML, CSS, React
2. Back-end proficiency: Node.js, email server, REST
3. Advanced tools: TypeScript, Webpack, AJAX, Axios → building an email client

## Course Structure

- **7 modules**, each with readings + programming exercises
- **No pre-defined deadlines**: demo exercises to TA when ready
- **Sequential prerequisites**: must complete current module before advancing
- **Demos**: delivered in groups of 6 exercises; must be able to explain all code
